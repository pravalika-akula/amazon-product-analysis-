# Amazon Product Analysis 

Analyzed 1000+ Amazon products to identify items using Python,Pandas, and Matplotlib.
## Key Findings 
- Products with 100+ customer reviews have an average rating of 4.1/5
- Top 5 products are rated above 4.5/5
- Price and review count have a weak correlation

  ## Tools Used
  - **Python**: Data cleaning and analysis with Pandas
  - **Matplotlib**: Data visualization
  - **Jupyter Notebook**: Analysis and reporting
  - **SQL**: Querying and prototyping
